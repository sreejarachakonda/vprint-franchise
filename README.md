# VPRINT Franchise Website

Modern responsive React + Tailwind CSS website for **VPRINT by ZAYVION Private Limited**, with Supabase integration for franchise lead capture.

## Tech Stack
- React (functional components)
- Tailwind CSS
- Framer Motion animations
- Lottie web-player animation support
- Supabase form storage

## Project Structure
```text
/src
  /components
  /pages
  /assets
  /hooks
  /utils
  App.js
  index.js
tailwind.config.js
```

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Create your env file:
   - Copy `.env.example` to `.env`
   - Add your Supabase values:
     - `REACT_APP_SUPABASE_URL`
     - `REACT_APP_SUPABASE_ANON_KEY`
3. Create table in Supabase:
   ```sql
   create table if not exists franchise_requests (
     id bigint generated by default as identity primary key,
     name text not null,
     mobile text not null,
     email text not null,
     profession text not null,
     address text not null,
     description text not null,
     created_at timestamp with time zone default now()
   );
   ```
4. Start local server:
   ```bash
   npm start
   ```

## Build
```bash
npm run build
```
